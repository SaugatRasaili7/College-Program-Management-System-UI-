# 🎓 College Program Management System (React + Vite)

## 📌 Project Overview

The **College Program Management System** is a web-based application developed using **React.js (Vite)**. This system is designed to help manage college events and programs efficiently by allowing users to add, view, and schedule events in a simple and user-friendly interface.

This project is developed as part of a **Grade 12 Software Engineering project**.

---

## 🚀 Features

* ➕ Add new events with details (Name, Date, Venue)
* 📋 View all available events
* 📅 Schedule events
* 🔍 Easy-to-use interface
* ⚡ Fast performance using Vite

---

## 🛠️ Technologies Used

* **Frontend:** React.js
* **Build Tool:** Vite
* **Language:** JavaScript (JSX)
* **Styling:** CSS
